== Description ==

A sidebar widget plugin which displays the latest price of Stellar Lumens.

== Installation ==

- Go to Plugins -> Add New
- Search for 'Display Stellar Lumens Price' and install the plugin
- Activate once installed

Once installed and activated, go to Widgets and drag the Stellar Price widget to a sidebar.
